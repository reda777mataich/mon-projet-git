# Comprendre le Version Control

Le version control est un système qui permet de suivre les modifications d’un projet dans le temps.

Il permet de :
- garder un historique des changements
- revenir à une ancienne version si nécessaire
- travailler à plusieurs sans perdre les modifications
- mieux organiser le développement d’un projet

Git est un outil de version control très utilisé pour gérer le code source.
